# ScrollSh
## Description
ScrollSh est un script permettant de faire scroller un text que vous choisisser avec une animation.

## Demonstration
![Alt Text](READMEFile/test.gif)

## Installation
### Download packages
```git 
git clone https://github.com/Thibault343/ScrollSh.git
```
### Script Intall
```bash
cd ScrollSh
./install.sh
```

## Usage
### words
```bash
scroll.sh [word]
```
### Phrases
```bash
scroll.sh "[phrases]"
```
### Exemples
```bash
scroll.sh Hello
```
```bash
scroll.sh "Hello World!"
```
