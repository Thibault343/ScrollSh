# ScrollSh
## Description
ScrollSh is a script for scrolling through text of your choice with an animation. 

## Demonstration 
![Alt Text](READMEFile/test.gif)

## Installation
### Download packages
```git 
git clone https://github.com/Thibault343/ScrollSh.git
```
### Script Intall
```bash
cd ScrollSh
sudo chmod +x ./install.sh
./install.sh
```

## Usage
### words
```bash
scroll.sh [word]
```
### Phrases
```bash
scroll.sh "[phrases]"
```
### Exemples
```bash
scroll.sh Hello
```
```bash
scroll.sh "Hello World!"
```
## Parameters
### colors
```bash
scroll.sh "Hello World!" -c red
```
